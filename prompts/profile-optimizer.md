# Profile Optimizer — AI Services Edition

## Purpose
Rewrite your bio for any platform (Upwork, LinkedIn, Fiverr, personal website) to attract AI services clients: automation, vibe coding, SaaS builds, AI consulting, workflow optimization.

## Input Required
Paste your answers to these 5 fields:
1. **PLATFORM**: [Upwork / LinkedIn / Fiverr / Website / X / Newsletter]
2. **TARGET CLIENT**: [e.g., "small business owners who need AI automation" / "course creators who need workflow systems" / "real estate agents who need listing automation"]
3. **SERVICE MIX**: [Which 2-3 services from your stack? e.g., "SaaS vibe coding + AI content systems + workflow automation"]
4. **PROOF POINT**: [One concrete result or credential — e.g., "built AllFours card game from zero using AI" / "published author" / "turned $242K debt into 3-hour operation"]
5. **RATE/POSITIONING**: [Bargain / Market / Premium — or specific rate if known]

## The Prompt

Run this with Claude/Kimi:

---
You are an elite personal brand strategist specializing in AI services positioning. Rewrite the following bio for [PLATFORM] targeting [TARGET CLIENT].

**Core Bio Assets:**
- Name: Shastrie Ramdhanie
- Brand: AI Sovereignty Movement
- Origin: Built from zero in Trinidad using AI-powered systems
- Constraint: $242K debt → 3-hour daily operation serving global audience
- Philosophy: Systems thinking, 1% daily improvement, consistency over intensity
- Assets: Published author "Hacking Your Mindset" (2023), Daily Newsletter, AllFours software
- Services: AI-powered business automation, "vibe coding" (building web apps/SaaS with AI), AI-assisted content creation, proposal writing, real estate listing automation, workflow optimization
- Differentiator: Doesn't just write code or copy — engineers systems that compound

**Target Positioning:**
- Service mix: [SERVICE MIX]
- Proof point: [PROOF POINT]
- Rate tier: [RATE/POSITIONING]

**Platform Rules:**
- If Upwork: Must include searchable title under 60 characters, hourly rate, split-benefit overview with bullet points, portfolio tagging strategy, and "Invite" CTA
- If LinkedIn: Must include headline under 220 characters, About section with story arc, Featured section hooks, and connection CTA
- If Fiverr: Must include gig title with keywords, package tiers (Basic/Standard/Premium), FAQ section, and "Order Now" urgency
- If Website: Must include hero section hook, problem-agitation-solution flow, social proof, and $27 book bundle CTA
- If X: Must include bio under 160 characters, pinned tweet structure, and thread hook template
- If Newsletter: Must include welcome email sequence, value proposition, and subscription CTA

**Output Format:**
1. **Title/Headline** (platform-optimized)
2. **Full Bio/Overview** (with platform-specific formatting)
3. **CTA** (platform-native conversion)
4. **SEO/Algorithm Tags** (keywords for discoverability)
5. **One-Line Pitch** (for DMs, intros, or cold outreach)

**Tone Rules:**
- No generic AI buzzwords ("leverage," "synergize," "unlock potential")
- Lead with constraint → transformation (the $242K/Trinidad/3-hour story)
- Every claim must tie to a specific outcome or mechanism
- Sound like a systems engineer who happens to use AI, not an AI tool user
- Use short, punchy sentences. No fluff.

## Usage Notes
- **Always verify identity** on Upwork before publishing (algorithm penalty for unverified)
- **Upload portfolio pieces** tagged to each service category (algorithm surfaces relevant work)
- **Price anchor**: Start at market rate, not bargain — your story is the credential, not the discount
- **Cross-reference**: Use the same one-line pitch across all platforms for brand consistency
- **Update monthly**: Refresh proof points as you ship new projects
